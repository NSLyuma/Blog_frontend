# [Блог](https://blog-frontend-nslyuma.vercel.app/)

### В проекте использовались:
#### Фронтенд:
- React, React Redux
- MUI
#### Бэкенд:
- Node.js
- Express
- MongoDB
- Insomnia

### Функционал:
- Регистрация и авторизация
- Просмотр всех статей
- Просмотр одной статьи
- Добавление, редактирование и удаление своих статей
